## Run Locally

Clone the project

```bash
  git clone https://github.com/Sameer-472/ICO-Dapp.git
```

Go to the project directory

```bash
  cd ICO-Dapp
```

Install dependencies

```bash
cd app
  npm install

cd
  npm install
```

Start the server

```bash
cd app 
  npm run dev
```
